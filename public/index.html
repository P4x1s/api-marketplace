5分钟快速开始

完全兼容OpenAI SDK，即插即用

# 安装 OpenAI SDK
pip install openai

# 使用 Aiko
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.aiko.com/v1"
)

response = client.chat.completions.create(
    model="mimo-v2.5",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

print(response.choices[0].message.content)

API 文档

快速开始

1. 注册账号并获取API密钥

2. 使用以下端点调用API：

POST https://dcwmum8geb.execute-api.us-east-1.amazonaws.com/dev/chat/completions

3. 在请求头中添加认证：

Authorization: Bearer your-api-key

请求格式

{
  "model": "mimo-v2.5",
  "messages": [
    {"role": "user", "content": "Hello!"}
  ]
}
